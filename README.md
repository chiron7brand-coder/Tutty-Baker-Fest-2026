# Tutty Baker Fest 2026 - Logistics & Site Map

**Event Planning Resources for Freeport, IL**

This repository contains all planning materials for the 2026 Tutty Baker Fest, including vendor layout, power/water assignments, inflatable placements, and printable maps.

## 📍 Overview
- **Location**: Exchange Street & surrounding blocks (Douglas to Adams), Freeport, Illinois
- **Key Features**: Food vendors, inflatables, live music, community booths, parade announcements
- **Power Sources**: Grey Box, alley power, lightpole, labeled Power A–H
- **Water Access**: Designated points (Water A–C)

## 📁 Files Included

- **`festival_vendors_full.csv`** – Complete vendor database (importable into QGIS, Excel, Google Sheets)
- **`Tutty_Baker_Fest_2026_Site_Map.pdf`** – Printable site map with symbols
- **`enhanced_festival_map.txt`** – Detailed ASCII/text layout
- **Logistics Documentation** – Block-by-block vendor assignments

## 🗺️ Site Map Highlights
- **Douglas to Exchange**: Food row with major vendors (Traveling Chef, 312 Beef, Dave's stands)
- **Exchange Street**: Command Center, concessions, petting zoo
- **Stephenson to Main**: Entertainment, food trucks, bands
- **Main & Adams**: Haack Pizza/Funnel Cake + final inflatables

**Inflatable Legend**:
- INF-01: Large Uncle Sam Walk-Through Arch
- INF-02: America 250 Star Arch
- INF-04: USA Firecracker Trio
- INF-05: Reserved/Sponsor
